# GAFFER — web

The Next.js 16 front end for GAFFER (static export → Firebase Hosting).

See the [root README](../README.md) for the full project: model, pipeline,
architecture, and how to run everything.

```bash
npm install
npm run dev        # http://localhost:3000  (live-refresh polls every 5s in dev)
npm run build      # static export → out/
npm run lint && npm run typecheck
```
